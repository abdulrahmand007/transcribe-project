## Transcribe AI Project

I developed an AI-powered audio transcription application using Python and Gradio. The project converts speech into text using Hugging Face speech recognition models through an interactive and user-friendly web interface.

The system supports audio preprocessing with Pydub to handle multiple audio formats and improve transcription performance. I also worked on fixing issues related to short audio transcription and grammar correction output to enhance the overall user experience.

### Features

* Audio-to-text transcription
* Simple Gradio web interface
* Support for multiple audio formats
* Audio summarization
* Grammar correction integration
  

### Technologies Used

* Python
* Gradio
* Hugging Face Transformers
* Groq API
* open router API
* Pydub

This project improved my practical experience in AI application development, speech processing, debugging, and integrating machine learning models into real-world applications.
